A Brief About Me

Hello!

Since my early childhood my Uncle gave me a set of COBOL floppy disks. Trying to understand that was very hard, and I never actually learned it, but from that point on the idea of programming was planted in my mind.

From learning html/css and PHP, to writing LUA code for the Playstation Portable (PSP) and creating applications in Visual Basic I have always strived to learn more. Whether it be through YouTube, StackOverflow or reading books. Programming is a skill and its a skill that cant be taken away from you. I in no way am a programming expert, but my philosophy is to never give up, because nothing is impossible.

I am currently learning Python, Java and C++. To aid in my learning I decided to create this website to document what I've learned and help others.
