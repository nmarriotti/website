Ruby is an object-oriented interpreted programming language developed in the 90's by Yukihiro Matsumoto. In this tutorial we'll cover creating a basic Hello World console program.

Assuuming that you already have Ruby installed, create a file hello.rb. All Ruby source files end with the .rb extension.


#! /usr/bin/env ruby

puts "Hello World!";

The first line of our script is needed to allow our script to be turned into an executable file and to print Hello World! to the console we used puts.

Open a terminal window and navigate to your hello.rb file and execute the following command to make the script executable.

chmod +x hello.rb

To execute your script you can type: 

ruby hello.rb
Hello World!

Since we made our script executable we could also run it by typing: 

./hello.rb
Hello World!
