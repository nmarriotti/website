Wouldn't it be nice if the Microsoft Windows command prompt could function just like the linux terminal? Well it can! In this tutorial we'll download Minimalist GNU for Windows and edit your systems environment variables to allow us to use commands such as ls in Microsoft Windows.

Required Downloads

Minimalist GNU for Windows

Installing MinGW

Open the MinGW installer and click Install. You can leave the default install location and other settings. Click continue. For this tutorial we will be installing in the C:\ drive.

Under the Basic Setup options, you will want to mark the following package for installation:

  • msys-base

Once those are selected click Installation>Apply Updates.

Edit PATH Environment Variable

Click the Windows Start and type in Environment. You will want to click Edit System Environment Variables.

Click Environment Variables near the bottom of the window.

Click the PATH variable and choose Edit. Using your MinGW install directory, add the following folder location to your PATH variable C:\MinGW\msys\1.0\bin and click OK.

Open your command prompt and starting using Linux terminal commands in Windows!
