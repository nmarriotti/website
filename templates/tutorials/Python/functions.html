Functions are used to break your program up into manageable organized pieces and allows for code to easily to be updated in the event changes are required. In this tutorial I'll give you a general understanding of how to create a function, call a function (execute the code in the function) and how to pass variables into a function.

Function Example

In this example we will create a variable called name and pass that variable into a function called greeting which will print out a greeting to the user.

Creating a Function

Here is our greeting function. All functions begin with def. Inside the parenthesis are the variables which will be passed in for your function to use. You can name these variables whatever you'd like.


def greeting(yourname):
	print("Hello {}, your name was passed into this function".format(yourname))

We created a function called greeting that accepts one variable yourname.

Calling the Function

Now that we have a function we are going to call it. When you call a function, you are telling the interpreter "Hey goto this section of code... execute it and then continue where you left off."


def main():
	name = "Nick"
	greeting(name)

Here's our required main function. We created our name variable and then called our greeting function passing in the name variable with it.

Output

Here is the complete code and it's output.


def main():
	name = "Nick"
	greeting(name)

def greeting(yourname):
	print("Hello {}, your name was passed into this function".format(yourname))

if __name__ == "__main__":
	main()


Hello Nick, your name was passed into this function
