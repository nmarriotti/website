Lists can store information in exactly what it sounds like... a list. This information can then be referenced based on an index.

Creating a List

In this example we are going to create a list of names.


names = []

There's our empty names list.

Add items to list

Now that we have our list created, lets add some information to it. To do that we use LISTNAME.append(ITEM).


names.append("Nick")
names.append("Bill")
names.append("Jim")
names.append("Sally")

Checking list size

To check the list size use len(LISTNAME)


print(len(names))

Output


4

Getting values

We had a list created and it contains information. Now we want to pull some of those values from the list. To get the values we will be referencing the index. For example, if we wanted the first value in the list that would be LISTNAME[0].

In this example I want to get Jim from the list.


print(names[1])

Output


Jim

Looping through list items

Inside of getting values individually using our example above, we could also use a loop to get each list item.


for item in names:
    print(item)

Output


Nick
Bill
Jim
Sally

Remove item from list

Let's say we want to remove Bill from the list. To do that it's very similar to how we add items; except this time we use LISTNAME.remove(item).


names.remove(1)

for items in names:
    print(item)

print("\nlist has {} values".format(len(names)))

Output


Nick
Jim
Sally

list has 3 values

So there you have it! That's how you can create, add, remove, and get items from a list in Python.
