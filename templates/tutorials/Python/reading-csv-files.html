In this tutorial we'll use the csv module to read a CSV file. The sample csv we'll be using contains the following information.

3,4
1,2
7,8
4,3
1,9
5,6

We'll start by creating a file object supplying the filename of our csv and set it to read-only.


f = open('sample.csv', 'r')

Next we'll use the csv module to create a csv reader object and pass our file to it. Our csv is comma-delimited so we can specifiy the delimiter the csv reader should use.


data = csv.reader(f,delimiter=',')

Now we're ready to print out the contents of our file. To do this we'll use a for loop.


for eachline in data:
	print eachline

Here is the output.

['3', '4']
['1', '2']
['7', '8']
['4', '3']
['1', '9']
['5', '6']
