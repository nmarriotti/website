In Python a docstring is a string literal beginning and ending in three apostrophes ''' typically used to comment blocks of code. This page was actually created using a docstring. In this tutorial we'll create a personalized message.


'''
This is a docstring.
'''

You can also use docstrings as a mask for a message, html, etc...


def main():
    message = '''    
Hello TO_NAME,
                
    Python is a widely used high-level programming language for general-purpose programming, 
created by Guido van Rossum and first released in 1991. An interpreted language, Python has 
a design philosophy which emphasizes code readability (notably using whitespace indentation 
to delimit code blocks rather than curly brackets or keywords), and a syntax which allows 
programmers to express concepts in fewer lines of code than possible in languages such as 
C++ or Java.

Thanks,
FROM_NAME
'''

Right now the message variable is set to our docstring. Check out the use of the unique TO_NAME and FROM_NAME. These words don't occur anywhere else in the string so we can use the print fnction to replace them with something else.


    print(message.replace("TO_NAME", "Jon Smith").replace("FROM_NAME", "Nick"))

The above print statement is replacing all instances of TO_NAME and FROM_NAME with Jon Smith and Nick leaving us with the following output.

Hello Jon Smith,

    Python is a widely used high-level programming language for general-purpose programming,
created by Guido van Rossum and first released in 1991. An interpreted language, Python has
a design philosophy which emphasizes code readability (notably using whitespace indentation
to delimit code blocks rather than curly brackets or keywords), and a syntax which allows
programmers to express concepts in fewer lines of code than possible in languages such as
C++ or Java.

Thanks,
Nick
