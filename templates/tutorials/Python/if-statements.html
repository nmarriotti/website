Conditional statements such as if, elif and else are very useful in determining whether a block of code should be executed. They control the flow of your program. Most people who have no programming experience at all have a good idea of what an if statement does. It executes code only IF a condition is True.

In this tutorial I demonstrate how to use these conditional statements to control the flow of a program.

Basics

Let's go over how to construct simple if statements.

If statement


if some_condition:
    # execute whats inside

if some_condition: will only execute if that variable is True.


running = True

if running:
    print running
    running = False

if running:
    print "Running is still true"

We initially set running = True. The program then checks if running:. Now remember the code inside the if statement will only be executed if our condition is True. It is! so we are going to print out the value of running and set it to False.

Then, on line 7, the program checks again to see if running is still True. Running is False so that code is not executed.

elif statement

Lets modify our example and add an elif(else if) statement. If the initial if statement is False it will then check the elif condition.


running = False
walking = True

if running:
    print "Running is still true"
elif walking:
    print "We're walking now"
    walking = False

So running is False. Is walking true? Yes! so print out We're walking now and set walking = False

Are you getting the hang of it? Lastly lets look at the else statement.

else statement

Else statements are executed only if all of the other conditions are False. Lets modify our example once more.


running = False
walking = False

if running:
    print "Running is still true"
elif walking:
    print "We're walking now"
    walking = False
else:
    print "We're not moving at all."

Since running is False and walking is False we just want to print out We're not moving at all..

Testing multiple conditions

What if you want to check multiple conditions? In that case you can use the following operators:

  • and
  • or
  • not

Here's a quick example


running = True
walking = True
sitting = False

if running and walking:
    print "Running is still true"
elif walking or running:
    print "One or more conditions are True"

if not sitting:
    print "Sitting is False"

Output

  • Running is still true
  • One or more conditions are True
  • Sitting is False
