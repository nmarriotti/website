Now lets get to the Python code to open and read that file.


    try: 
        fh = open("test.txt", "r")
        for line in fh:
            print(line)
    except:
        print("Unable to open file.")

I added some error handling to display an error if the file doesn't exist. We set fh to the file we want to open and used a for loop to grab each line in the file. But wait...
