Python is an object-oriented programming language. Objects are created from classes, which are essentially blueprints. Classes are comprised of all for a particular object. In this tutorial we'll learn how to create a basic class, take a look at constructor, work with modules, create objects and use some of our objects methods.

For this tutorial we will be creating a Book class. Before we begin coding we need to think of some of the traits a book has.

  • Title
  • Author

Creating a Class

Begin by creating a new file called Book.py in your current working directory and add the following code.


class Book():

Constructors

A constructor is a method containing code that will be executed immediately after instantiating (creating) an object. For now let's set our book's title using the constructor so you can see how it works.


    def __init__(self, title):
    	self.title = title

To create our constructor we call the __init__, or initialize method and pass in self and the title. Since our constructor now requires a title parameter we will need to be sure to set the title when we create our object later in the tutorial.

What is self?

Passing in self means that we are referring to a class variable and not an instance variable. An instance variable is a variable contained only inside our method.

Class Methods

Now we need to create some methods to set and get the title and author of our book object.


    def getTitle(self):
        return self.title

    def setAuthor(self, author):
        self.author = author

    def getAuthor(self):
        return self.author

For our get methods we are simply returning the variable being sure to always use self followed by the variable name and for the set methods we are setting our variable equal to the variable name in the method parameters.

Save your Book.py file.

Importing a Class

Create a new file main.py. The first thing we need to do is import our class. 


from Book import Book

What we are saying in the above statement is... from the file Book.py import the Book class. If our Book.py file was located in a folder called classes our import statement would be:


from classes.Book import Book

Now that we have our class imported we can create objects with it.

Creating an Object

Remember, since our class constructor requires a title we need to pass in a book title when we create our object. To create an object we can do something like this...


    book1 = Book("The Cat in the Hat")

Using Class Methods

To use our class methods we can use book1.METHODNAME(). Lets go ahead and set the author of our first book object.


    book1.setAuthor("Dr. Seuss")

Now lets use some print statements and our get methods to return our books title and author.


    print(book1.getTitle())
    print(book1.getAuthor())

Complete Code


# Book.py
class Book():
    def __init__(self, title):
        print("Book object created.")
        self.title = title
    
    def getTitle(self):
        return self.title

    def setAuthor(self, author):
        self.author = author

    def getAuthor(self):
        return self.author


# main.py
from Book import Book

def main():
    book1 = Book("The Cat in the Hat")
    print(book1.getTitle())
    book1.setAuthor("Dr. Seuss")
    print(book1.getAuthor())

if __name__ == "__main__":
    main()

That covers the basics of creating and using classes in Python. You can create as many objects of that class as needed, because remember, classes are just blueprints for objects. If you have any questions please be sure to leave a comment below.
