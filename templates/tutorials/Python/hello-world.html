The Hello World application is the most basic tutorial for any programming language. In this tutorial we will learn how to setup a basic Python script and output the string "Hello World!" to the console."

Lets begin by creating a file: HelloWorld.py 


def main():
    print("Hello World!")

if __name__ == "__main__":
    main()

In the above code, we defined a method called main. def main():. A method is simply a block of code that can do something. Inside that method we are telling Python to print the string "Hello World!" to the console.

if __name__ == "__main__": This statement is used so that this script can be run directly.


Save the file and in the console type: python HelloWorld.py. You should see output similar to the example below.


				$ python HelloWorld.py

				Hello World!
