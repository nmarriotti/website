So you'd like to learn Python, but don't know where to start? In this tutorial I walk you through downloading and installing Python.

What is Python?

Python is a high-level object-oriented programming language. It's simple, easy to learn syntax emphasizes readability and allows for easier maintenance.

Required Downloads

You may sometimes think newer is always better. When it comes to Python that is not always the case. Not all packages are compatible with the newest version, version 3.6 and for that reason I would recommend Python 2.7.

Get Python 2.7 (Windows)

Installing

Open the installer and choose whethere to install it for all users or just your account.

Choose an install location. The default location, usually C:\Python27 is fine.

On the next screen you can select what is installed. The defaults are fine but make sure you click Add python.exe to PATH and mark it for installation. When something is added to the PATH that means you can execute that program from within any directory in your filesystem

Python 2.7 will now be installed. Click Finish. You are now ready to move onto the next tutorial where we will create our first Python program.
