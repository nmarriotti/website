Variables are containers which hold information of multiple data types whether it be an integer, double, float, string, or a boolean. Python makes it very easy to declare variables as the datatypes are automatically determined by the interpreter.

Declaring a Variable

Here are some examples of variables using the different data types.


myNumber = 10
myString = "Nick Marriotti"
myFloat = 19.99
myBoolean = True

So we just created a few variables. Since the python interpreter automatically determines the data type you just set your variable name equal to a value.

Type Casting

Let's try to combine myString and myNumber in a print statement.


myNumber = 10
myString = "Nick Marriotti"

print(myString + " has " + myNumber + " dollars")

Our desired output should be Nick Marriotti has 10 dollars. Below is our output.

print(myString + " has " + myNumber + " dollars") TypeError: cannot concatenate 'str' and 'int' objects

The error is telling us that we cannot combine myNumber into a string because it is currently an integer. We can fix it by temporarily converting it to a string though! Let's modify our code.


myNumber = 10
myString = "Nick Marriotti"

# Method 1
print(myString + " has " + str(myNumber) + " dollars")

# Method 2
print("{name} has {money} dollars".format(name=myString, money=myNumber))


Nick Marriotti has 10 dollars
Nick Marriotti has 10 dollars

It worked! Take note of our second method of writing the print statement. You can convert variables to different data types by wrapping your variable in any of the following functions:

  • str()
  • int()
  • float()
  • double()


So there you have it! That's how variables work in Python :)
