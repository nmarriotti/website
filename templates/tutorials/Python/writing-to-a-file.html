Writing to a file in Python is very easy and requires very little code.


    writer = open("testfile.txt", 'w')
    writer.write("This is sent to the output file.\n")
    writer.close()

In the code above we called the open function specifying the filename we want to create and used w for write mode and stored that into a writer variable. Next all you do is call the write() function and include the statement you want to send to the file. When finished with wthe file be sure to close it using the close() function.
