Lambda functions, or anonymous functions, are similar to normal functions, except they are not created with the def keyword and do no contain a function name. In this tutorial I'll show you how to create a basic lambda function and show you how they they compare to your typical functions.

Here is a simple lambda function that will multiply a number by two.


    f = lambda x: x*2
    print f(10)

We created a variable f and set it equal to the result of x*2. In the next line you can see we want to print the result of f(10). It's basically the same thing as saying...


    def f(x):
        return x*2

    print f(10)

Both functions return the same result 20. The choice is yours on how you want to create your functions. Lambda's just require a little less code.
