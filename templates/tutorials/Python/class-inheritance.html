When you create a class in Python you can use inheritance, which allows your class to inherit all traits and methods from another class.

Creating the Super Class

Our Pet class will hold the name of our Pet and the type of pet we have as well as some methods to return the name and type.


class Pet:
    def __init__(self, name, type):
        self.name = name
        self.type = type

    def getName(self):
        return self.name
    
    def gettype(self):
        return self.type

    def __str__(self):
        return "{} is a {}".format(self.name, self.type)

Using Inheritance

When you define your class, inside the quotations you simply put the name of the class you want to inherit from. The inherited class then becomes known as the parent or super class.

In this example we will create a Dog class and set it to inherit everything from the Pet class.


class Dog(Pet):
    def __init__(self, name, breed):
        Pet.__init__(self, name, "Dog")
        self.breed = breed

    def getBreed(self):
        print(self.breed)

When we create an object of our Dog class the constructor for the Pet class will not automatically be called, because the Dog constructor takes precedence so we have to call our Pet class constructor manually.


        Pet.__init__(self, name, "Dog")

Creating Objects

Now that we have our classes built it's time to create a Dog object and see the inheritance in action.


    Buddy = Dog("Buddy", "Golden Retriever")
    print(Buddy)
    Buddy.getBreed()

    Max = Dog("Max", "Pug")
    print(Max)
    Max.getBreed()

In the above example we created an object Buddy of the Dog class, set the name to Buddy and set the type to Golden Retriever.

When we run our code we get the following output.

Buddy is a Dog
Golden Retriever
Max is a Dog
Pug
