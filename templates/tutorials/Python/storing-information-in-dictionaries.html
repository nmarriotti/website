Dictionaries are very similar to an array, except values are set for keys. In this tutorial we will be creating a school grades dictionary and by the end of the tutorial you'll have learned how to create a dictionary, set values to keys, sort, update values and print the information

Creating a Dictionary


    grades = {
           "Math": "A",
        "Science": "B",
        "History": "D",
        "English": "F"
    }

There is our grades dictionary. The keys are set to the classes: Math, Science, History and English. Keys are to be wrapped in double quotations followed by a semicolon. We then set the grades: A, B, D and F

.

Printing Dictionary

Now that we have our dictionary created lets print out the information. You can print a single key, value pair or you can print everything out using a for loop like we did in the previous tutorial with lists.

Single Record


    print(grades["History"])

Output


D

All Records


    for key, value in grades.items():
        print("{} grade is {}".format(key, value))

Output


Science grade is B
English grade is F
Math grade is A
History grade is D

The above for loop is grabbing every key and value from the dictionary and printing it to the screen.

If you want to print just the values. You can modify your for loop.


    for grade in grades:
        print(grades[grade])

Output


D
B
F
A

Sorting

You may have noticed that when we printed our grades dictionary out the keys were not in alphabetical order. To do that we can use the sorted() function.


    for key, value in sorted(grades.items()):
        print key, value

Output


English F
History D
Math A
Science B

Updating a value

So what if someone improved their grade in lets say... History class. Here's how you would update the history grade.


    grades["History"] = "C"

Now we can print out our dictionary again to see the changes.


    for key, value in grades.items():
        print key, value


Science B
English F
Math A
History C

Complete Example

Here is the code used for this tutorial.


def main():
    grades = {
           "Math": "A",
        "Science": "B",
        "History": "D",
        "English": "F"
    }

    for key, value in grades.items():
        print("{} grade is {}".format(key, value))

    
    # Print specific key
    print(grades["History"])


    # Print just the values
    for grade in grades:
        print(grades[grade])

    # Sort 
    for key, value in sorted(grades.items()):
        print key, value

    # Update value
    grades["History"] = "C"
    for key, value in grades.items():
        print key, value

if __name__ == "__main__":
    main()

Congrats! You just learned how to use dictionaries in Python!
