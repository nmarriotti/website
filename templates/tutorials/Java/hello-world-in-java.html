Java is one of the most popular programming languages. It's source code is compiled down to bytecode that is ran from the Java Virtual Machine. Before we get started make sure you have the Java Runtime Environment (JRE) and Java Development Kit (JDK) installed.

When working with Java, all classes and functions MUST be wrapped in curly braces and all statements MUST end with a semicolon.

Let's get started! Create a file Hello.java and add the following code:


public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

Notice how we have public class Hello... All Java files are essentially a class, and the class name is typically uppercase and the same name as the file itself. We'll get into working with classes and objects later. First we need to learn the basic outline for a program.

We now have public static void main(String[] args) This is where your program begins and is required for it to run. If you've worked with C or C++ its essentially the same thing as the int main function.

To print strings, we want to use System.out.println(). The println() function will put whatever string it contains on its own line.

To compile your program, open a console or terminal and navigate to where your file is saved and execute the following command:

javac Hello.java

Now that our program is compiled lets run it with: 

java Hello

You should now see our string, "Hello World" printed to the screen.

Hello World

Congrats, you just wrote, compiled and ran your first Java program!
