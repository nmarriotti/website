In this tutorial we'll create a program which asks the user to enter their name. The name will be stored into a string variable and printed back to the user. To do this we will need to create a Scanner object

.

Create a file Input.java and add the following code.


import java.util.Scanner;

The java.util.Scanner package must be imported in order to capture user input.

Next we need to create our class. Remember class names must be the same name as the file you created.


public class Input {



}

If you remember from my Hello World tutorial, we need to add our main function, so lets add that inside our class now.


import java.util.Scanner;

public class Input {
    public static void main(String[] args) {
        

    }
}

The rest of the code in this tutorial will go inside our main function.

Creating Scanner Object

Since we imported the Scanner package we can now create a Scanner object.


Scanner s = new Scanner(System.in);

Declaring Variable

Next we need to create a variable that the users input will be stored in. Since we are asking the user for their name we will use a string variable.


String name;

Using Scanner object

We're almost done. Using the System.out.print() function, we need to ask the user a question.


System.out.print("Enter your name: ");

To prompt the user to enter input we need to use our Scanner object and call the next() function since our name is a string. If we wanted to store a whole number that the user entered we would use nextInt(). 


name = s.next();

At this point if we ran our program, the user would be prompted to enter their name and Scanner would wait until something is typed in and for the ENTER key to be pressed.

Printing the Variable

Finally, we need to print out the users name. Printing out a variable is very easy.


System.out.println("You entered: " + name);

Compile and Run

Our program is now finished. All we need to do now is compile and run it. Open up a terminal or console and type:

javac Input.java

Hopefully you didn't receive any errors compiling. If all went well run the program by typing:

java Input

Here is the output from the program:

Enter your name: Nick
You entered: Nick

Complete Code


import java.util.Scanner;

public class Input {
    public static void main(String[] args) {
        Scanner s = new Scanner(System.in);
        
        String name;
        
        System.out.print("Enter your name: ");
		name = s.next();

        System.out.println("You entered: " + name);    
    }
}
