In the last tutorial we learned how to create a class and create/use an object of that class. Now we are going to get into inheritance. Inheritance allows you to create an object of one class, known as the childclass, but also inherit, or use the variables and methods from another also known as the parent or super class.

For my example I will show you how inheritance works by creating basic bank account classes.

Account Class

The account class is going to be our super class. It will hold the account name and total amount in the account.

Create a file Account.java and add the following code:


public class Account {

    private String name;
    private long amount;

    public Account(String name, long amount) {
        this.name = name;
        this.amount = amount;
        this.setAmount(amount);
    }

    void deposit(long amount) {
        this.amount += amount;
    }

    void setAmount(long amount) {
        this.amount = amount;
    }
    
    String getName() {
        return name;
    }

    long getAmount() {
        return amount;
    }

}

Looking at our class you can see the constructor public Account() requires a name and amount to be passed in. It also has methods to:

  • Deposit money
  • Set amount
  • Get account name
  • Get amount in account

    • That does it for our Account class. Lets move on to our next class.

    CheckingAccount Class

    To have our class inherit from the Account class we have to use the extends keyword when declaring our class.

    Create a file CheckingAccount.java and add the following code:

    
public class CheckingAccount extends Account {

    public CheckingAccount(long amount) {
        super("Checking", amount);
    }
    void withdraw(long amount) {
        setAmount(getAmount() - amount);
    }
}

    The CheckingAccount class has a single method to withdraw money.

    What is super()?

    super() is to be put in the constructor of the child class and is used to run the constructor in the parent class. We have to use super(), because that's where our name and amount variables are contained.

    Calling Methods

    If your parent and child classes contain the same method names, the method contained inside the child class will take precedence and run, vice the method in the parent class.

    Main method

    Create a file Main.java and add the following code:

    
public class Main {
    public static void main(String[] args) {

        CheckingAccount a = new CheckingAccount(495);  /* create object with initial value of $495 */
        System.out.println("Name: " + a.getName() );   /* Print out the account name */
        System.out.println("Amount: " + a.getAmount() ); /* Print amount */
        a.withdraw(150); /* withdraw $150 */
        System.out.println("Amount: " + a.getAmount() ); /* Print amount again */
    }
}

    Testing it out

    Compile and run your program. Sample output is below.

    Name: Checking
Amount: 495
Amount: 395
