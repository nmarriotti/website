Essentially everything in Java is a class and an object. If you've never messed around with object-oriented programming languages, by the end of this tutorial you should have a fairly good idea of to create a class and an object of a class in Java.

Classes define all the variables and methods for a particular object. For example, lets say you wanted to make an address book program. You could create a contact class and create a new object of that class which holds a particular persons contact information. In this tutorial we'll create something similar.

Create a file Main.java and lets starting get the main method down. Remember class names MUST be the same name as the source file.


public class Main {
    public static void main(String[] args) {


    }
}

Save the file and create a new file Contact.java. Similar to what we did for our previous file we need to define our class.


public class Contact {
	

}

Defining Class Variables

Common practice is to make all variables private, meaning they are only accessible from within the same class. Lets create a variable.

public class Contact { private String name; }

Constructor

The constructor is a method with the same name as the class. It contains blocks of code that will be executed automatically after an object of that class is instantiated. Think of it as an initializer. You can have it set all your defaults. For now we'll just have it print a statement so you can see how it works.


public class Contact {
	
	private String name;

	public Contact {
		// Constructor
		System.out.println("Object created!");
	}

}

Getters and Setters

To prevent classes from being able to directly access variables it's best to create getter and setter methods. Methods are blocks of code which complete a certain task.

Getter methods are commonly used to return a variables current value while Setter methods are used to set a variables value.


    public void setName(String name) {
        this.name = name;
    }

    public void getName() {
        System.out.println(this.name);
    }

There we have two methods. One that sets the value of our name variable and one that gets the value of our name variable and prints it out. this.name tells the compiler that we are referring to the class variable called name.

Final Contact class

The complete Contact.java class is below. All that's left to do now is create an object in our main method.


public class Contact {
    String name;

    public Contact() {
        System.out.println("Object created!");
    }

    public void setName(String name) {
        this.name = name;
    }

    public void getName() {
        System.out.println(this.name);
    }
}

Creating an Object

Heading back to our Main.java file, we need to create an object of our contact class. Here is the outline for creating an object.

ClassName objectName = new ClassName()

So to create our object it will be:


Contact c = new Contact()

Calling Class Methods

Now we need to set the persons name for our object. We want call class methods using objectName.methodName(). 


c.setName("Nick Marriotti");
c.getName();

Complete Main.java

Your completed Main.java file should look like the one below.


public class Main {
    public static void main(String[] args) {
        Contact c = new Contact();
        c.setName("Nick Marriotti");
        c.getName();
    }
}

Testing it all out

The last thing we need to do is compile and run our program.

javac Main.java

java Main.java

Object created!
Nick Marriotti

Our constructor was automatically called and printed Object created!. We set our name variable using our setter method and printed it using our getter method. If you'd like to have the constructor set our name variable, you can modify the following sections of code.


        Contact c = new Contact("Nick Marriotti");
        c.getName();


    public Contact(String name) {
        System.out.println("Object created!");
        setName(name);
    }

    
That concludes the tutorial on creating classes and objects. Later on in the tutorial series we'll get into more advanced classes.
