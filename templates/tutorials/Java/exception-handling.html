Exceptions are errors in a program which cause it to terminate unexpectedly. These exceptions must be handled to allow the program to continue running. Exception handling in Java is achieved using try/catch statements.

In this tutorial we'll write a little program which will try to divide a number by zero. Division by zero is not allowed and will throw an exception. We'll handle this error so our program can continue running.


public class Main {
    public static void main(String[] args) {
            
            int x = 1/0;
            System.out.println(x);
            System.out.println("Hi there!");

    }
}

When we compile and run the above program we get the following output.

Exception in thread "main" java.lang.ArithmeticException: / by zero at Main.main(Main.java:4). We can see that the name of the exception is ArithmeticException.

Lets implement our try/catch statements so we can handle this error.


public class Main {
    public static void main(String[] args) {
        
        try {
            int x = 1/0;
            System.out.println(x);
        } 
        catch (ArithmeticException e) {
            System.out.println("Error: " + e);
        }

        System.out.println("Hi there!");

    }
}

Error: java.lang.ArithmeticException: / by zero
Hi there!

After implementing exception handling we caught the error using the catch statement and printed an error to the user and our program continued executing without abruptly terminating.
