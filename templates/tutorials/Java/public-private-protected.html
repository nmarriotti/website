If you are just beginning Java it is important to understand the differences between the access modifiers and this is important when it comes to working with variables and methods for your classes. We'll get into classes in a later tutorial. For now I'll just explain what the modifiers are.

Access Modifier Description
public Variables and methods marked public are visible to everything meaning any object can access it.
private Only visible within the class it's contained in.
protected Visible to the subclass.

Keep following my tutorials and we'll get into using these modifiers in some classes so you'll be able to get a better understanding of each.
