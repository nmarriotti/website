In Java an ArrayList is something you can use to create dynamic arrays which you can add and remove values to whenever necessary. When creating an array you must also declare the data type or object class defining the type of information the list will contain. In this tutorial we will create an ArrayList of strings and look at how we can sort the data in the ArrayList.

Import Package

Before we can create an ArrayList object we must include the required package.


import java.util.ArrayList;

Creating ArrayList

The basic outline for creating an ArrayList is:

ArrayList<DATATYPE> LISTNAME = new ArrayList<DATATYPE>();

Lets go ahead and create an ArrayList of strings called MyList.


ArrayList<String> MyList = new ArrayList<String>();

Adding Items

To add values to our ArrayList we can use the ArrayList class method add(). Lets go ahead and add three values now.


MyList.add("Orange");
MyList.add("Blue");
MyList.add("Purple");

Removing Values

To remove values from an ArrayList you can call the remove() method specifing the index where the value is located. The example below would remove the value Purple from MyList.


MyList.remove(2);

Get List Size

Just like adding and removing information from a list there is also a method to get the size or total number of values contained in the ArrayList.


MyList.size();

Getting Value

To get a value from the list you can use the get() method and give it the index for the value you want to retrieve. The example below would retrieve the value located at index 0.


MyList.get(0);

Looping through items

To print out everything in our list we will be using a for loop. The example below is basically saying for each item in MyList.


for(String item : MyList) {
    System.out.println(item);
}

Example Program

Here is a sample program using an ArrayList and all the methods we talked about in this tutorial.


import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        
        int size; /* Will store size of list */

        //Create an ArrayList of string values 
        ArrayList<String> MyList = new ArrayList<String>();  

        // Add values to the List
        MyList.add("Orange");
        MyList.add("Blue");
        MyList.add("Purple");

        // Remove item from list
        MyList.remove("Purple");

        // Set our list size
        size = MyList.size();

        // Get value from list
        System.out.println("Value at index 0 is: " + MyList.get(0));

        // Print out the list size
        System.out.println("List size: " + size);

        // Print everything in our list
        for(String item : MyList) {
            System.out.println(item);
        }
    }
}

Here is the output from the above program.

Value at index 0 is: Orange
List size: 2
Orange
Blue
