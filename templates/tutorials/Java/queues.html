Queues are essentially just a list of items that use a First In First Out (FIFO) principle. An example I like to use for this is a printer queue. When a bunch of people are trying to print some documents all jobs are added to the printers queue and the printer prints out in the order the documents were sent to the printer.

In this tutorial I'll show you how to create and use a queue object.

Required Packages

The first thing you'll need to do is import the Queue and LinkedList packages.


import java.util.Queue;
import java.util.LinkedList;

Creating the Queue

For this example we'll create a queue to hold Strings.


Queue<String> q = new LinkedList<String>();

Add Method

To add values to our queue we'll use the add() method.


q.add("Job1");
q.add("Job2");
q.add("job3");

Size Method

Now that we added some values to our queue lets check the size.


int size = q.size();
System.out.println("The size is: " +size);

3

Peek Method

Since queues use the FIFO principle we cannot reference values based on an index. The peek() method is used to grab the first item in the queue.


String s = q.peek();
System.out.println("The first value is: " + s);

The first value is: job1

Element Method

Similar to the peek() method, the element() method is used to grab the first item in the queue and will throw a NoSuchElementException if the queue is empty.


String s = q.element();
System.out.println("The first value is: " + s);

The first value is: job1

Poll Method

The poll() method will grab the first item in queue and then remove it. If the queue is empty then null is returned.


String s = q.poll();
System.out.println("The first value is: " + s);
System.out.println("The size is: " + q.size());

The first value is: job1
The size is: 2

Remove Method

The remove() method returns the first item in the queue and removes it. If the queue is empty a NoSuchElementException is thrown.


String s = q.remove();
System.out.println("The first value is: " + s);
System.out.println("The size is: " + q.size());

The first value is: job2
The size is: 1

Complete Code

Here is a sample program using all the methods explained above.


import java.util.Queue;
import java.util.LinkedList;
public class Main {
    public static void main(String[] args) {
		Queue<String> q = new LinkedList<String>();

        q.add("job1");
        q.add("job2");
        q.add("job3");

        String s = q.peek();
        System.out.println("The first value using peek() is: " + s);

        System.out.println("The size is: " + q.size());

        s = q.element();
        System.out.println("The first value using element() is: " + s);

        s = q.remove();
        System.out.println("The first value using remove() is: " + s);
        System.out.println("The size is: " + q.size());

        s = q.poll();
        System.out.println("The first value using poll() is: " + s);
        System.out.println("The size is: " + q.size());
    }
}

The first value using peek() is: job1
The size is: 3
The first value using element() is: job1
The first value using remove() is: job1
The size is: 2
The first value using poll() is: job2
The size is: 1
