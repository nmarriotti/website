At this point we have only been using System.out.println to print to the console. Another way you can do it is using System.out.format() which gives you more options when it comes to formatting your ouput.


        String myString = "This is a string";
        int myInt = 10;
        float myFloat = 49.3f;
        char myChar = 'C';

        System.out.format("%s%n", myString);
        System.out.format("%d%n", myInt);
        System.out.format("%f%n", myFloat);
        System.out.format("%c%n", myChar);
        System.out.format("%.2f%n", myFloat); /* 2 Decimal Places */

Here is the output from the above code.

This is a string
10
49.299999
C
49.30

This should get you started with the basics. For more information regarding the System.out.format function refer to the official documentation.
