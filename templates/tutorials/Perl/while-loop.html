While loops will execute code as many times as needed until a condition is false. Perl while loops are no different than they are in the C programming language.


$i = 0;

while ($i < 10) 
{
	print("i is $i\n");
	$i += 1;
}

When we execute the above code we get the following result:

i is 0
i is 1
i is 2
i is 3
i is 4
i is 5
i is 6
i is 7
i is 8
i is 9
