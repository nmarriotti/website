Arrays are used to store multiple values within a single variable. These values can then be referenced based on an index. The first value stored in an array will always be at index 0. Unlike regular variables that are declared using the dollar sign $, arrays are declared using the at sign @.


@myArray = (1, 2, 3, 4);

In the code above we created an array called myArray and loaded it with the values 1, 2, 3 and 4.

If you wanted to pull the third value from the array we would use the 2nd index.


@myArray[2]

To get the size of an array (number of values stored), we can do something like this.


scalar(@myArray)

Lets use a for loop that we learned in the previous tutorial to print out all the values stored in our array.


@myArray = (1, 2, 3, 4);

for($index=0; index<scalar(@myArray); $index++) 
{
	print("@myArray[$index]\n");
}
