Getting user input in Perl is very simple. In this tutorial we'll ask the user to enter some information and store the input in a variable.


my $color;

print("What's your favorite color? ");
$color = <STDIN>;

print("Your favorite color is $color\n");

In the above code, we created a variable called color and asked the user what their favorite color is, set our color variable equal to what was entered into the standard input STDIN, and finally printed out what the user entered.

When we preview our code notice the blank line following our final print statement. Why is this happening?

What's your favorite color? Blue
Your favorite color is Blue

The blank line is due to the standard input adding a newline terminator \n when the ENTER key is pressed. We can fix this using chomp on our variable which will remove the newline terminator. Here is the modified code.


my $color;

print("What's your favorite color? ");
$color = <STDIN>;
chomp $color;

print("Your favorite color is $color\n");

What's your favorite color? Blue
Your favorite color is Blue
