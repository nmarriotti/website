When using double quotes around strings you can use variable interpolation to replace the variable name with the actual value stored in the variable.

There are only a couple ways to include variables in strings. We can concatenate using the dot (.) operator or use interpolation and include the variable name directly inside the double quoted strings. Here are a few examples.


$food = 'steak';

print 'Fred ate a $food \n';  # WRONG. Will print $food and \n
print 'Fred ate a ' . $food . "\n"; # RIGHT. Uses concatenation
print "Fred ate a $food \n"; # RIGHT. Uses interpolation.

Using interpolation, in my opinion, is the most efficient way to include variables in strings. It also requires less code compared to using concatenation. When using single quotes, what is included inside the single quotes is printed except a backslash followed by a single quote (if you want a single quote printed), which is why our newline characters\n are wrapped in double quotes.
