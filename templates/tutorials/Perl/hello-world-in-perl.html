Perl is a family of high-level, general-purpose, interpreted, dynamic programming languages that was originally designed for text processing. After working with other programming languages I feel the Perl syntax is a little ugly, but that's not to say Perl isn't a great language. So lets get started creating our first Perl program.

Create a file hello.pl. All Perl programs will end with the .pl extension.

The first line of our script will allow our program to be turned into an executable program.


#! /usr/bin/env perl

Similar to the include and import statements that are seen in other languages, Perl uses the use. The strict and warnings lines we are going to add to our program tell the Perl interpreter to check for programming errors.


use strict;
use warnings;

Now we are ready to write the code to print text to the console.


print "Hello World!\n";

All lines in Perl must end with a semicolon and there are no specific rules on spacing or indentations, although properly spaced and indented syntax can greatly improve readability.

Save your file and we'll now use chmod to convert our program into an executable file. Navigate into the directory where hello.pl is located and execute the below command.

chmod +x hello.pl

To run our program we can type:

./hello.pl
Hello World!

We can also run our program using:

perl hello.pl
Hello World!

That concludes my Hello World tutorial in Perl. In the next tutorial we'll get into working with variables.
