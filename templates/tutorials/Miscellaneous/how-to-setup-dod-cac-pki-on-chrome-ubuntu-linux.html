In this tutorial we'll setup our Common Access Card (CAC) to work in Google Chrome running on Ubuntu Linux and install the DoD root certificates.

Setting up the CAC Reader

First thing you need to do is download cackey.

32-bit

Execute the following command.

wget http://cackey.rkeene.org/download/0.7.5/cackey_0.7.5-1_i386.deb && sudo dpkg -i cackey_0.7.5-1_i386.deb && sudo apt-get install -f

64-bit

Execute the following command.

sudo mkdir /usr/lib64 && wget http://cackey.rkeene.org/download/0.7.5/cackey_0.7.5-1_amd64.deb && sudo dpkg -i cackey_0.7.5-1_amd64.deb && sudo apt-get install -f

Google Chrome Setup

Execute the following commands.

sudo apt-get install libnss3-tools

Close Chrome.

32-bit

modutil -dbdir sql:.pki/nssdb/ -add "CAC Module" -libfile /usr/lib/libcackey.so

64-bit

modutil -dbdir sql:.pki/nssdb/ -add "CAC Module" -libfile /usr/lib64/libcackey.so

Open Chrome to try going to a CAC-enabled site.
