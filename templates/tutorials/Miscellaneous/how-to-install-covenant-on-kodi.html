Kodi is an open-source media center that lets you install addons. Covenant is very similar to Exodus, one of the most popular addons. Covenant gives you the ability to search for any tv show or movie and stream it for FREE! In this tutorial I'll guide you through installing the Covenant addon.

Required Downloads

If the download link below does not work, you can grab the Kodi installer from the official website

Download Kodi 17.1

Once the download is complete, open the installer and proceed to install Kodi.

Installing Exodus

  • Run Kodi
  • Click settings
  • Click System Settings
  • Click Add-ons
  • Enable Unknown Sources
  • Go back to settings and click File Manager
  • Click Add Source
  • Type fusion as the name for the media source
  • For the media source URL put http://mediarepos.net/kodistuff and click OK
  • Go back to the home screen and click Add-ons
  • Click the box icon
  • Click Install from zip file
  • Select kodistuff > RepoColossus
  • Choose Install from repository
  • Click Colossus repository
  • Click Video add-ons
  • Click Covenant
  • Click Install

The Covenant add-on will now be downloaded and installed. To open Covenant, goto the home screen and click Add-ons.

Have fun watching your favorite TV shows and movies! :)
