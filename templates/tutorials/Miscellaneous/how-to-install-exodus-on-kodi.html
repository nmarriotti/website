Install Covenant
Kodi is an open-source media center that lets you install addons. One of the most popular addons is Exodus. Exodus gives you the ability to search for any tv show or movie and stream it for FREE! In this tutorial I'll guide you through installing the Exodus addon.

Required Downloads

If the download link below does not work, you can grab the Kodi installer from the official website

Download Kodi 17.1

Once the download is complete, open the installer and proceed to install Kodi.

Installing Exodus

  • Run Kodi
  • Click settings
  • Click System Settings
  • Click Add-ons
  • Enable Unknown Sources
  • Go back to settings and click File Manager
  • Click Add Source
  • Type fusion as the name for the media source
  • For the media source URL put http://fusion.tvaddons.ag and click OK
  • Go back to the home screen and click Add-ons
  • Click the box icon
  • Click Install from zip file
  • Select fusion > kodi-repos > english
  • Scroll down and select repository.exodus-XXXX.zip
  • Choose Install from repository
  • Click Exodus repository
  • Click Video add-ons
  • Click Exodus
  • Click Install

The Exodus add-on will now be downloaded and installed. To open Exodus, goto the home screen and click Add-ons.

Have fun watching your favorite TV shows and movies! :)
