The strtok()function is used to split strings that are separated by a delimiter. For example suppose we have the following string.

This,is,a,comma,delimited,string

Using the strtok() function we can pull out each word that is separated by the comma. The declaration is char *strtok(char *str, const char *delim). It returns a char and takes in a char string, and delimiter as parameters. You will need to include string.h to use the function.

Declare Variables


char myString[] = "This,is,a,comma,delimited,string";
char delim[2] = ",";
char *token;

We created three char variables. One contains the string we want to parse, the second is the delimiter and the last is a pointer which will point to each separated word from our string. Now lets use the strtok() function to start parsing this string out.


token = strtok(myString, delim);
printf("%s", token); /* example */

This

When we run the code you can see it's working, but it only pulled out the the word This. To parse the entire string we need to use a loop.


while(token != NULL) {
    printf("%s\n", token);
    token = strtok(NULL, delim);
}

Complete Code


#include <stdio.h>
#include <string.h>

int main() {
    char myString[] = "This,is,a,comma,delimited,string";
    char delim[2] = ",";
    char *token;

    token = strtok(myString, delim);
    printf("%s", token);

    while(token != NULL) {
        printf("%s\n", token);
        token = strtok(NULL, delim);
    }

    return 0;
}
