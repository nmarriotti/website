When you execute your program via the command line, you can also also arguments, or additional criteria such as specifying input and ouput files your program should use. In this tutorial I will give you an example of how to create a program to capture these arguments and output them all to the screen.

Command line arguments are stored in an array. We haven't touched on arrays yet, but we will in the next tutorial. Lets say we need to create a program that reads an input file, manipulates the data in some way and writes the output to a new file.

In your int main() function modify it to match what is below.


int main(int argc, char* argv[]) {
	
}

As you can see our int main now takes in an integer variable argc and a char pointer array argv[]. What is this stuff?

argc is a variable that counts the number of command line arguments passed in the terminal or command-line at runtime.

argv[] is an array that stores each argument passed. Each value is stored based on an index. The first command line argument will always be your program executable.

Now lets say to run our program we type ./run but we want to now specify our input and output file as well. To do that type something like this...


./run inputfile.txt outputfile.txt

If we ran our program with the above arguments argc would hold the value of 3 and since argv[] is an array we can reference each value like so:

  • argv[0] = ./run
  • argv[1] = inputfile.txt
  • argv[2] = outputfile.txt
