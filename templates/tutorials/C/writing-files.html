To create files and write to them we will need to include stdio.h and a file pointer defined as:


FILE *fp;

Now we need to set the filename and the mode that we want to open it in.


fp = fopen("test.txt", "w+");

Using our file pointer fp we are using the fopen() function and telling it to create a file called test.txt and open it in w+ mode. Here is a listing of the available modes you can choose from.

Mode Description
r Open an existing file to read.
w Opens a text file for writing. If it does not exist then a new file is created.
a Opens a text file for writing in appending mode.
r+ Opens a text file for both reading and writing.
a+ Opens a text file for both reading and writing. It creates the file if it does not exist. Reading will start from the beginning, but writing can only be appended.

Now lets start writing some strings to our file. Here are two ways you can do it:


fprintf(fp, "This is testing for fprintf\n");
fputs("This is testing for fputs\n", fp);

Both fprintf and fputs do the same thing and they both require the file pointer fp to be passed as a parameter.

When finished writing to the file we have to close it.


fclose(fp);

When we run our program you will see a test.txt file is created and contains our two strings.

This is testing for fprintf
This is testing for fputs

Here is the complete code.


#include <stdio.h>

int main() {

        FILE *fp;
        fp = fopen("test.txt", "w+");

        fprintf(fp, "This is testing for fprintf\n");
        fputs("This is testing for fputs\n", fp);

        fclose(fp);

}
