Imagine you are creating a program and want to ask the user for a filename. By now you should know how to use puts() and printf() to output text to the screen, but how we can grab data that the user has typed from the keyboard? To do this we'll be using the scanf() function.

First, lets create a variable that we can use to store our input.


char filename[80];

This char called filename can hold up to 79 characters. Now lets ask the user for the name of the file.


printf("Enter a filename: ");

Time to get some data from the user. Add the following line of code.


scanf("%79s", filename)
printf("You entered: %s", filename);

The scanf() function is used to capture input from the keyboard. The 79 tells us that only 79 characters of what the user types will be stored into the filename variable and the s tells the compiler that what the user is entering show be stored as a string. The next printf() is just grabbing the stored data and printing it back to us.

Here is a demo of the program running.

Enter a filename: test.txt
You entered: test.txt

Want some practice storing user input? Try creating some variables with different data types and modifying the scanf and print statements.
