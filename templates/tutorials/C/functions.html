Functions help break you code up into manageable pieces. You can pass variables into a function and you can return values. In this tutorial we'll create two basic functions, go over writing the function prototype and calling the functions. When I say call what I mean is run the function.

Prototype

The compiler compiles the code sequentially from top to bottom. When you call a function, the compiler has to be able to say "Oh ok this is where your calling your function". So we have to give the compiler a heads up on what our function is before we use it.

A function prototype is just you telling the compiler this is the return type of my function and it takes in these parameters. Prototypes can go inside a header file or can be places at the top of your code right before the int main().

Here are the prototypes for the functions we'll use in this tutorial.


void saySomething();
int doSomething(int);

Functions that will not return a value should be declared as void. If you want to do some calculations and return a value you will need to declare your functions as either int, float, double, char, etc.... You'll see what I mean later in the tutorial.

Looking at our doSomething(int) you can see inside the parenthesis it has int. That means it requires an integer value to be passed in when the function is called and since we have int at the beginning of the declaration it is going to return an integer value.


int main() {
		int number;

        saySomething();
        number = doSomething(6);
        printf("Number is", number); 
        return 0;
}

Here is our int main(). We created an integer variable and are calling our saySomething() function and then we are calling our doSomething(6) function. Notice how we also passed in the integer value of 6. Our program is going to jump to each function, executes its code and return where it left off. Lets declare our functions now.

Function Declarations

Below your int main() is where we are going to declare our functions and add the code that we want each to run.


void saySomething() {
        printf("Hello there!\n");
}

int doSomething(int x) {
        x = x * 2;
        return x;
}

The first function is just printing a statement. The doSomething(int x) function is multiplying the number we passed in (6) by 2 and returning the value of x which is stored in the number variable.

Time to compile and run our program to see these functions in action.

Hello there!
Number is 12

When we pass variables into functions we are essentially creating a copy of them. This takes up more memory. In the next tutorial I'll show you how we can use pointers to avoid our functions from taking up more memory from creating these copies.
