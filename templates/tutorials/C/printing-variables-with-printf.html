Referring to the last tutorial where we printed "Hello World!" to the screen. What if we wanted Hello World! to be placed in a variable and print out just the variable? In this tutorial I'll be explaining the basics of the printf function and how to print variables to the terminal.


char name[5] = "Nick";
int apples = 3;

Those are the variables we're going to use. In this example we want to print Hello there! Nick has 3 apples. Lets add our printf function.


printf("Hello there! %s has %i apples.\n", name, apples);

Notice the %s and %i. The %s is referring to a string variable and the %i refers to an integer variable. We then ended the line using \n and then put the names of our variables. The first one is the string and the second is an integer.

Go ahead and compile and run your program. You should see the following output:


Hello there! Nick has 3 apples.

Here is the complete code used for this example. Feel free to modify and play around with it.


#include <stdio.h>

int main() {

        char name[5] = "Nick";
        int apples = 3;

        printf("Hello, %s has %i apples.\n",name, apples);
        return 0;
}

Remember, the puts function will NOT print out variables. For printing variables you'll want to use printf. This tutorial was pretty quick and didn't go into much depth. But it should be enough to get you started. Leave a comment and let me know what you thought.
