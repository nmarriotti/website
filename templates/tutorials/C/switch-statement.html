Switch statements are very similar to if statements. For if statements your saying: "If this is equal to that then do this, else do that..."

Here is the basic outline for a switch statement.



        switch(expression) {
                case constant-expression: 
                        //do this
                        break; /* optional */
                case constant-expression:
                        //do this
                        break; /* optional */
                case constant-expression:
                        //do this
                        break; /* optional */
                default: /* optional */
                        //else do this
        }

You can have as many case statements as you'd like. What this is saying is that depending on what the main expression is, execute the code inside the corresponding case statement.

In this tutorial we are going to ask the user to enter a number 1-3 and use the switch statement to determine what the user entered.


        int option;

        printf("Select an option 1-3: ");
        scanf("%i", &option);

        switch(option) {
                case 1: 
                        puts("option is 1");
                        break;
                case 2:
                        puts("option is 2");
                        break;
                case 3:
                        puts("option is 3");
                        break;
                default:
                        puts("Invalid option!");
        }

Here is what the code is doing:

  • Defined integer variable called option
  • Asked the user to enter an option
  • Store the input into the memory address of our option variable
  • Depending on the value of option execute the corresponding case statement

Note: If a case statement does not include a breakIt will execute every case statement following it.

Lets run the program.

Select an option 1-3: 2
option is 2

Select an option 1-3: 4
Invalid option!
