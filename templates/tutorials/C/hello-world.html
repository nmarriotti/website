So you want to start learning the C programming language? Creating a Hello World program is usually the first step and that's what I'm going to show you how to do. Lets get started!

General Outline

All C programs will contain a general outline consisting of includes, which allow your program to use other functions and an int main() function. The int main() is where your program begins.

Begin by creating a file hello.c and add the following code:


#include <stdio.h>

int main() {
	//PUT CODE HERE
	return 0;
}

Print text to terminal

Now that we have our general outline, lets make the string Hello World! print to the screen. To do this we will be using the puts() command. 


#include <stdio.h>

int main() {
	puts("Hello World!");
	return 0;
}

What is puts()?

puts() is a function to print unformatted text to the screen. You cannot include variables when using puts(). We will get more into variables and printing formatted text to the screen in a later tutorial.

Compiling

Computers read 1's and 0's. To make our computer understand our program we first need to compile it to convert it to machine language. Enter the following command to compile your program:


gcc hello.c -o hello

Wondering what the command is doing? Well... we are telling the C compiler (gcc) to compile hello.c and save the machine language ouput -o as hello. Note: If you are on Windows, you will be presented with a .exe file after compiling.

Now it's time to run your program:


./hello


Hello World!

Congrats, You just wrote your very first C program! Although the program doesn't do a whole lot right now, we'll continue to build on this example and learn more about C in the upcoming tutorials.
