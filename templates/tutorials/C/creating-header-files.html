Header files in C contain the .h extension. One of the most common is the stdio.h header file. Including these files essentially copies and pastes their contents so that we can use them in our code. In this tutorial we'll be creating a very basic program that adds two integers together and add them together, printing out the result to the user.

Create a file add.h and add the following function:


int addNumbers(int x, int y) {
        return x+y;
}

Inside of our header file we are creating our function to add the two integers together which will return the answer. That's it for our header file!

Now create a file called example.c and add the following includes:


#include <stdio.h>
#include "add.h"

Notice a difference between the two? Header files contained in the C library are not wrapped in double quotes. Header files wrapped in double quotes tell the compiler that it is a local file and should look in the current directory.

Now we can use our addNumbers() function located inside add.h. Paste in the remainder of our program.


int main() {

        int num1, num2, result;

        printf("Enter a number: ");
        scanf("%d", &num1);
        printf("Enter another number: ");
        scanf("%d", &num2);

        result = addNumbers(num1, num2);
        printf("Result: %d\n", result);

        return 0;
}

Congrats! You just created your own header file and used its function addNumbers(). Header files can help your code stay organized and separate from your program. If you have any questions please leave a comment.
