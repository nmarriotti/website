Imagine we're reading in a CSV file and have to manipulate the data. We know that each value in the row should goto a specific variable, but what do all of these values have in common? They all belong to a specific row. Using a struct we can group all of the rows values together and reference them based on what we named our struct.

Sorry if I confused you, but hopefully by the end of this tutorial you'll have a better understanding of what I mean. In this tutorial we are working with recipes. All recipes have a few things in common.

  • Recipe name
  • Description
  • Prep Time
  • Number of servings

Lets create a struct to group all of these things together for a specific recipe.


        struct recipe {
                const char *name;
                char *description;
                int prepTime;
                int servings;
        };

There's our struct. All it does is allocate memory to hold our variables for each recipe. Let's create a recipe using our struct and load it with some values.


        struct recipe spaghetti = {"Spaghetti", "It's super delicious", 20, 4};

What we did here is create Spaghetti using our recipe struct and loaded each variable with a value. Now we're ready to print the values contained in our spaghetti recipe. To do that we'll use a printf statement and call spaghetti.VARIABLENAME.


        printf("Recipe: %s\nDescription: %s\nPrep Time: %d\nServings: %d\n",
               spaghetti.name, spaghetti.description, spaghetti.prepTime, spaghetti.servings);

We can reference all of the variables we loaded in our spaghetti recipe struct like so:

  • spaghetti.name
  • spaghetti.description
  • spaghetti.prepTime
  • spaghetti.servings

    • typedef

    To reference our struct by a different name we can use typedef. Lets say we want to reference our struct using the word recipe. Checkout the modified code.

    
        typedef struct {
                const char *name;
                char *description;
                int prepTime;
                int servings;
        }recipe;

        recipe spaghetti = {"Spaghetti", "It's super delicious", 20, 4};

    Lets run our program and see what output we get.

    Recipe: Spaghetti
Description: Some delicious spaghetti
Prep Time: 20
Servings: 4
