In this tutorial we will open the .txt file we created from the last tutorial and output it to the console.


#include <iostream>
#include <fstream>
using namespace std;

int main() {
    
    // Reading
    string line;
    ifstream inputfile("example.txt");
    
	if(inputfile.is_open()) {
        while(getline(inputfile, line)) {
            cout << line << endl;
        }
        inputfile.close();    
    } else {
        cout << "Unable to open file\n";
    }
    
    return 0;
}

Explanation

Line 8: Created a string variable named line. This variable will hold the contents of each line we read from the file.

Line 9: Created an input stream ifstream> object named inputfile and set the file we want to open.

Line 11: We used inputfile.is_open() because we only want to get the contents of the file if the file opened successfully.

Line 12-14: Loop through each line in our inputfile object and store each line in the string called line and output it to the console.

Line 15: Close the file.

Line 16-18: If the file didn't open then display an error message.
