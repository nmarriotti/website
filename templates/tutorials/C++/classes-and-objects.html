When you think of a bank account what do they all have in common? All accounts fall under a specific bank, they all have a balance, an account type and transactions. When you purchase something with your debit or credit card money is deducted from your account and when you make a deposit, money is added. You may be asking yourself what's your point?

Well... I am trying to explain what a class is. Classes define objects and all of their traits and actions. So you could create a People class... all people have names, hair color, eye color, height, weight, etc... See what I'm saying?

Public and Private

It is good practice to make all class variables private, meaning they can't be accessed directly from the int main() and can only be accessed from within the class itself. With private variables we will be creating getter and setter functions to update our variables.

Methods or functions of the class should be public so that we can call them outside of the class.

In this tutorial I am going to demonstrate a very basic Bank Account class and you'll be able to see how we will update our variables and call our class functions.


class Account {
    
private:
    float balance;
    string bankName;
    
public:
    Account(string name, float balance) {
        cout << "This is the constructor" << endl;
        setBalance(balance);
        setBankName(name);
    }
    
    void setBalance(float newBalance) {
        balance = newBalance;
    }
    
    void setBankName(string name) {
        bankName = name;
    }
    
    float getBalance() {
        return balance;
    }
    
    string getBankName() {
        return bankName;
    }
    
};

So here is our bank account class. Notice how the class name Account is capitalized. Although its not super important that it be capitalized. It is common practice to do so.

Constructor

Take a look at our first class function. Notice it's called Account, the same name as our class itself and takes in a bank name and a balance? This is our constructor. The constructor is executed immediately after an object is created and in our case we want to update the bank name and balance as soon as an object is created.

Getters and Setters

We also have a few functions to update our variables bankName and balance with each function beginning with either get or set.

Creating an Object of our Account class

So we have our class created. Let's use it. 


int main() {
    Account newAccount("First Bank", 100.00);
    
    return 0;
}

What we are saying in the above code is... Create an object called newAccount of the Account class and set the bank name to First Bank and the balance to 100.00.

Testing it out

Go ahead and execute your program. Notice how the only output you see is:


This is the constructor!

Let's call some of our class functions and see what our variables are set to and update our balance.

Using Class Functions

To get the balance of our newAccount object you would type: newAccount.getBalance() and to get the bank name type newAccount.getBankName().

Want to update the balance? Just call the setBalance function and pass in the new balance. newAccount.setBalance(5000).

Here is the final output for our code. The constructor was the first thing executed and set our bank name and initial balance. We then updated our balance.


This is the constructor
Bank name: First Bank
Current Balance: 100
New Balance: 5000

Complete Code

Here is the complete code. Feel free to copy and paste it and play around with it to get a better understanding of how classes and objects work.


#include <iostream>
using namespace std;

class Account {
    
private:
    float balance;
    string bankName;
    
public:
    Account(string name, float balance) {
        cout << "This is the constructor" << endl;
        setBalance(balance);
        setBankName(name);
    }
    
    void setBalance(float newBalance) {
        balance = newBalance;
    }
    
    void setBankName(string name) {
        bankName = name;
    }
    
    float getBalance() {
        return balance;
    }
    
    string getBankName() {
        return bankName;
    }
    
};

int main() {
    Account newAccount("First Bank", 100.00);
    cout << "Bank name: " << newAccount.getBankName() << endl;
    cout << "Current Balance: " << newAccount.getBalance() << endl;
    
    // Update the balance
    newAccount.setBalance(5000);
    cout << "New Balance: " << newAccount.getBalance() << endl;
    
    return 0;
}

This tutorial was meant to only cover the basics. There is a ton more to learn when it comes to object oriented programming. I hope I have helped you get a better understanding of what a class is, how to create an object, constructors, and how to use getter and setter functions.
