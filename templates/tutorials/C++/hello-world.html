In this tutorial we will learn how to write a basic Hello World console program. This tutorial assumes you already have a C++ IDE installed and a compiler configured.

Begin by creating a file hello.cpp

Copy and paste the following code.


			  #include "iostream"
				using namespace std;

				int main() {
					cout << "Hello World!\n";
					return 0;
				}

So what is that code doing? Well first we tell C++ that we want it to include all of the functions included in the iostream library by writing #include "iostream"

using namespace std is optional but it allows use to write I/O statements with a little less code. For example:


				// Without using namespace std
        std::cout << "Hello World!\n";

int main() is a required function. It is where your "main" program begins. Without it your program will not compile properly. We will talk about functions and return statements such as return 0; in a later tutorial.

cout << "Hello World!\n"; is what we want to display on the screen and we tell the compiler that by using the cout command. All strings must be placed in double quotations as well. The \n is a line terminator (newline) and tells the compiler that the next statement we output needs to be on the next line.

Now let's compile and run our program. You should get the following output.


						Hello World!
