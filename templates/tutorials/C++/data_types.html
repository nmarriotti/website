Looking back at the last tutorial where I mentioned packaging up kitchen applicances into a box labeled "Kitchen Applicances", let's talk about how this analogy applies to C++.

Imagine you had some whole numbers, decimal numbers, some words (text), some characters (example: 'B') and maybe a true/false variable. When creating variables to hold these different types of information there are Data Types that must be declared first.

Basic Data Types

Type Keyword Purpose
String string Used to hold text values
Boolean bool True/False
Character char Example 'M' or 'F' for a gender
Integer int Whole numbers
Floating point float Decimal number with 7 digits or precision.
Double floating point double Decimal number with 15 digits of precision
Valueless void Used primarily to declare functions that return no values

Sample Program

Here is a sample program that uses some of the basic data types described above and the resulting output. There's a lot more to learn about data types. For more information click here.


	#include "iostream"
	#include 

	using namespace std;

	int main()
	{
	    int age = 10;
	    float money = 50.00;
	    double pi = M_PI;
	    string name = "Nick Marriotti";
	    char gender = 'M';
	    bool isMale = false;

	    cout << "Age is " << age << endl;
	    cout << "Money is " << money << endl;
	    cout << "Pi is " << pi << endl;
	    cout << "Your name is " << name << endl;
	    cout << "Gender is " << gender << endl;

	    if(gender == 'M') {
	        isMale = true;
	    }

	    if(isMale) {
	        cout << "You are male!";
	    }

	    return 0;
	}

Sample Output
