With loops we can run groups of statements any number of times as long as a condition is true.

There are three main types of loops:

  • while
  • do While
  • for

    • while Loop

    If you'd like to repeat blocks of code an unknown number of times while a boolean condition is true, you'll likely be using a while loop.

    
int i = 0;

while(i <= 10) {
    cout << i << endl;
    i += 1;
}

    In the above example we created an integer variable and set it to 0. The code inside the while loop is executed until i is greater than 10.

    do While

    The do While loop is very similar to the while loop, however, with a do while loop, the loop will always execute one time. Have a look at the example below.

    
int i = 20;

do {
   cout << "Inside the loop" << endl;
} while (i <= 10);

    We defined an integer variable i and set it equal to 20 and tell the loop to print Inside the loop as long as i is less than 20. Now we know i is set to 20, but the statement is going to print to the screen one time anyway.

    for

    You would most likely use a for loop if you know exactly how many times you'd like to execute a block of code.

    
for(int i=0; i<5; i++) {
    cout << i << endl;
}

    In this example we set i equal to 0 and we want to loop until i is greater than or equal to 5 and every time it loops we increment i to 1 by using i++
