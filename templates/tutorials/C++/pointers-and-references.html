Pointers are variables which point to to an address in memory where another variable is held. They are especially useful when it comes to memory management and passing variables into functions. In this tutorial I'll give you a basic understanding of how pointers and references work.

Declaring a pointer

Here is the basic outline for declaring a pointer.


DataType* VariableName = new DataType;

Using the template above lets say we want to create an integer pointer called PointsToAnInt. Here is how you would do that.


int* PointsToAnInt = new int;

Using the Pointer

So now that we have created a pointer lets make it point to something.


int* PointsToAnInt = new int;
int age = 28;

PointsToAnInt = &age;

As you can see we created an integer variable called age and set it equal to 28. We then told our pointer to point to the memory address that our age variable is stored at using the & ampersand

Let's look at some output from the code below to get a better idea of what's going on here.


int* PointsToAnInt = new int;
int age = 28;

PointsToAnInt = &age;

cout << "PointsToAnInt points to memory address of " << PointsToAnInt << endl;
cout << "PointsToAnInt holds a value of " << *PointsToAnInt << endl;

Output


PointsToAnInt points to memory address of 0x61ff18
PointsToAnInt holds a value of 28

So as you can see adding the * asterisk will return our integer value that is held at the memory location PointsToAnInt is pointing to.

References

Now that we have an idea of what a pointer is lets get into references. Think of a reference as the memory location and not the actual value of the variable. When you create functions and pass variables to them you are essentially creating a copy of that variable. If the data being passed into the function is rather large it could be using up precious memory. To conserve some memory we can pass a reference to the variable instead.


PointsToAnInt = &age;

In this statement we told PointsToAnInt to reference the address in memory that age is stored at. Ampersand(&) is how you can reference variables.

Reference variable in a Function

So here we have a program that will calculate the users age based on the year they were born. We have created a getAge function and we will be passing it the reference of what the user types, which will be the year variable.


#include <iostream>
using namespace std;

// Initialize function
int getAge(int& year);

int main() {
    
    int year = 0;
    int* pointer = new int;
    
    cout << "Enter your birthyear: ";
    cin >> year;
    
	pointer = &year;
	
    cout << "\npointer is stored at " << &pointer << endl;
    cout << "pointer points to value of " << *pointer << endl;
    cout << "You are " << getAge(*pointer) << endl;
    
    cin.get();   
    return 0;
    

}

int getAge(int& year) {
    return (2017 - year);
}

Explanation

Looking at our function getAge you can see we referenced the memory location that our year variable is stored at and when we called our function we passed in the actual integer value using *pointer

Here is the output from the above code.


Enter your birthyear: 1988
pointer is stored at 0x61ff18
pointer points to value of 1988
You are 29
