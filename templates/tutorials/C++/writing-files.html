In this tutorial we will create a .txt file and write some strings to it.


#include <iostream>
#include <fstream>
using namespace std;

int main() {
    
    // Writing
    ofstream myfile;
    myfile.open("example.txt");
    myfile << "We are going to write this string to the file\n";
    myfile.close();
    
    return 0;
}

Explanation

Line 8: Created an output filestream object named myfile.

Line 9: Opened a file called example.txt using our myfile object.

Line 10: Sent a string into example.txt.

Line 11: Closed the file.
