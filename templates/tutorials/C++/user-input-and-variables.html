In the last tutorial we output a string to the console, but what about user input? How can we capture that and store it? It's actually very easy and we'll do so using a variable.

What is a variable?

Imagine your moving and are packing up some boxes. You start in the kitchen and label the box "Kitchen Appliances". Seeing that label would tell you that only kitchen appliances should go in that box right? Variables are very similar to a labeled box. They're containers that can only hold certain types of information, aka Data Types which we will learn in the next tutorial.

Lets get coding! Create a file input.cpp


#include "iostream"
using namespace std;

int main() {

	string name;

	cout << "What is your name? ";
	cin >> name;

	cout << "Hello, " << name << "!" << endl;

	return 0;
}

What's that code doing?

We created a string variable by typing string name;. A string is a data type treated as text such as "Hello World!"

cout << "What is your name? "; We ask the user a question.

cin >> name; Here we used the cin command with the arrows >> pointing towards our name variable. So with this line we are getting the value that is stored in the name variable.

cout << "Hello, " << name << "!" << endl; Here we output the word "Hello" and we want to get what was stored in our name variable. We add an exclamation point and end the line endl;

Testing it out!

Save your code, compile, and run it. Type your name in when asked and press Enter. You should see the following output:


What is your name? Nick
Hello, Nick
