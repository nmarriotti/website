Functions are used to break your code up into organized pieces. They can also take in variable values, pass by value, or they can update variables, pass by reference, and return values. We'll look at examples of each in this tutorial.

Let's say you were creating a menu application with options for the user to choose from; instead of copying the menu code and pasting it where it's needed, you can just call a create menu function.

Using functions also enables you to only have to update code in one spot rather than many, making it a heck of a lot easier if you need to update your code down the road.

Functions that DO NOT return a value

The menu we are going to create is only going to print to the screen so we will define our function as void and we'll name it menu.


void menu() {
	cout << "1. Option" << endl;
	cout << "2. Option" << endl;
	cout << "3. Option" << endl;
	cout << "4. Option" << endl;
}

So there's our menu function. Notice there's nothing inside the parenthesis, because we don't need to take in any variables. Now lets use it.


#include "iostream"
using namespace std;

// Declare our function
void menu();

int main() {
	
	// Call the function
	menu();
	
	return 0;
}


void menu() {
	cout << "1. Option" << endl;
	cout << "2. Option" << endl;
	cout << "3. Option" << endl;
	cout << "4. Option" << endl;
}

Line 5: We have to declare our function before our int main(), because our function is defined below the main function.

Line 10: Here we CALL our function by using the function name. This tells the compiler to jump down to our function and execute the code inside it.

Line 16-21: This is our function and all we want it to do is print out our menu.

Functions that DO return a value

When creating a function that returns a value, the function must be defined with its return data type.

In this example we are going to add two whole numbers together and return the answer. Since we're dealing with whole numbers we'll use an integer data type.


#include "iostream"
using namespace std;

// Declare our function
int add();

int main() {
	
	// Call the function
	int i = add();
	
	return 0;
}


int add() {
	int result = 5 * 10;
	
	return result
}

Just like our menu function, we declared our function above our int main, but this time we are setting an integer variable equal to the return value of our function, which is our result variable.

Pass by Value


#include "iostream"
using namespace std;

// Declare our function
int add(int, int);

int main() {
	
	// Call the function
	int i = add(5, 2);
	
	return 0;
}


int add(int x, in y) {
	int result = x * y;
	
	return result
}

Line 5: We are passing in two variables, so inside the parenthesis we add the data type for each variable.

Line 10: We pass in the the values 5 and 2.

Line 16: Since we passed in 5 and 2 when we called the function, int x is now equal to 5 and int y is equal to 2.

Line 17: Set result variable equal to x*y or 5*2.

Line 19: return the value of our result variable, which is 10, back to where this function was called.
