What is an array?

Arrays are simply variables that can hold multiple values of the same data type and are referenced via a zero-based index. They can be static, meaning they can only hold a set number of values; or dynamic, and hold as mainly values as needed.

What is an index?

Think of a bookshelf. If you wanted to grab the third book from the shelf, you would just count them 1..2..3.. and grab the book right? Arrays store values just like the books on a shelf example, but the first book starts at index zero. So if you wanted to grab the third book that would be at index 2. If your confused don't worry, I'll explain more in the examples below.

In this tutorial I'll provide example of both static and dynamic arrays, how to declare them, how to initialize them and how to get the values stored inside.

Static Arrays

Here is the outline for declaring a static array.


DataType ArrayName [Length] = {values};

As mentioned above, static arrays can only hold a set number of values. Let's create an array of type int called myNumbers that can hold 5 values and load it with some values.


const int ARRAY_LENGTH = 5
int myNumbers [ARRAY_LENGTH] = {5, 7, 10, 35, 41};

Explanation

We created a constant integer variable named ARRAY_LENGTH and set it to 5 and declared our array of type int called myNumbers with a length of 5 and set the values as 5, 7, 10, 35, 41.

Dynamic Arrays

What if we weren't sure how many values our array should hold? In this case we will create a dynamic array. We will create our myNumbers again, but to make it a dynamic size we will use a vector.

Here is the outline for declaring a dynamic array.


vector<DataType> ArrayName [Length] = {values};

Let's create it it now


#include <vector>

    vector<int> dynArray (5);

    // Set our 5 values
    dynArray[0] = 5;
    dynArray[1] = 7;
    dynArray[2] = 10;
    dynArray[3] = 35;
    dynArray[4] = 41;

Explanation

1. When using vectors, you must add #include .

2. we created a vector of type int named dynArray with an initial size of 5.

3. Added 5 values to our array.

How to add another value

If you need to add another value to your dynamic array you would use the push_back() function 


dynArray.push_back(55);
dynArray.push_back(100);

Conclusion

Arrays are very usual if you need to store multiple values of the same type. I hope by now you have a basic understanding of how to declare them, add values and get values. Here is a sample C++ file demonstrating both static and dynamic arrays. Feel free to play around with it.


#include <iostream>
#include <vector>

using namespace std;

int main()
{

    // Static
    const int ARRAY_LENGTH = 5
    int myNumbers [ARRAY_LENGTH] = {1, 2, 3, 4, 5};
    cout << "Value at index 2 is " << myNumbers[2] << endl;


    // Dynamic
    vector<int> dynArray (5);

    // Set our 5 values
    dynArray[0] = 5;
    dynArray[1] = 7;
    dynArray[2] = 10;
    dynArray[3] = 35;
    dynArray[4] = 41;

    // Get the initial size
    cout << "Size of dynArray is " << dynArray.size() << endl;

    // Add two more values
    dynArray.push_back(55);
    dynArray.push_back(100);

    // Get the new size
    cout << "Size of dynArray is " << dynArray.size() << endl;



    return 0;
}
