To control the flow of a program you can use IF statements. They can be used to determine if a block of code should be executed or not. Below is the basic structure of an IF statement.


		if (condition) {
			  statement(s)
		}

What is a condition?

Think back to math class. Remember the greater than or equal to, not equal to, etc... Operators such as those are usually contained inside these if statements to determine what block of code should be executed. Below is a chart of the different conditional operators you can use in C++.

Operator Purpose
== true if and only if left operand is equal to right operand
!= true if and only if left operand is not equal to right operand
> true if and only if left operand is greater than right operand
>= true if and only if left operand is greater than or equal than right operand
<= true if and only if left operand is less than or equal than right operand

Logical Operators

In C++ the logical operators are && meaning AND, || meaning OR, and ! meaning NOT.

Sample Program

I put together a program along with the output to demonstrate how the IF statements are used and how the various conditions and logical operators used. Create a project, compile it and test it out for yourself.


	#include "iostream"

	using namespace std;

	int main()
	{
	    int x = 10;
	    bool y = false;

	    if(x == 10) {
	        cout << "x is 10" << endl;
	    }

	    if(x <= 20) {
	        cout << "x is LESS THAN OR EQUAL TO 20" << endl;
	    }

	    if(x >= 11) {
	        cout << "x is GREATER THAN OR EQUAL TO 11" << endl;
	    }

	    if(x != 9) {
	        cout << "x is NOT EQUAL to 9" << endl;
	    }

	    if(!y) {
	        cout << "y is FALSE" << endl;
	    }

	    if(x == 1 || x == 10) {
	        cout << "x is EQUAL to 10" << endl;
	    }

	    if(x < 20 && x > 0) {
	        cout << "x is LESS THAN 20 and x is GREATER THAN 0" << endl;
	    }

	    return 0;
	}
