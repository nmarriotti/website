Welcome to my first tutorial on C++. In this tutorial we will be downloading MinGW which includes the C++ compiler and Code::Blocks, a free open source Integrated Development Environment (IDE) and setting up our first project.

Required Downloads

Minimalist GNU for Windows
Code::Blocks IDE

Installing MinGW

Open the MinGW installer and click Install. You can leave the default install location and other settings. Click continue

Under the Basic Setup options, you will want to mark the packages shown below for installation:

  • mingw32-base
  • mingw32-gcc-g++
  • mingw32-gcc-objc
  • msys-base

Once those are selected click Installation>Apply Updates. When the installation has finished it's time to install Code::Blocks.

Installing Code::Blocks IDE

Open the Code::Blocks setup.exe file and keep clicking next through the installation until it is complete. You can leave all the defaults. You should then see a screen that shows the detected compilers. It should show (GNU Compiler) as detected.

That's it! Now let's start our first project.

Creating our first C++ Project

Open Code::Blocks. Click File > New > Project and select Console Application

Click next and when asked what language you wish to use, select C++. Give your project a name and we are ready to run the default Hello World program Code::Blocks has setup for us.

One the left of the screen expand Sources and double click your main.cpp file.

You will learn more about the Hello World program in the next tutorial. For now let's build and run this program. At the top of the screen click Build > Build and Run or press F9.

Congratulation! You just setup the C++ compiler, IDE and ran your first program!
